[Превью с решением](https://evgiss.github.io/the-droplet/)

## Функционал

- 'резиновый' адаптив
- hover-эффекты, в своответствии с макетом
- кликабельное меню
- на разрешении 768px и ниже - бургер меню

При разработке использовалось:

- gilp сборка
- scss
- jquery

## Для запуска

Установка:

```sh
npm install
```

Режим разработки:

```sh
npm run dev
```

Режим сборки:

```sh
npm run build
```
